# mtm6201-final
Web Portfolio
Project Overview
This is a fictional digital design agency website consisting of three pages: Home, Blog, and About. The site was built using HTML, CSS, and Bootstrap 5, based on a Figma design provided by the instructor.
Development Process
The structure of each page was analyzed from the design mockups first, then the HTML was built section by section, followed by writing the CSS. All three pages share a single CSS file, with CSS variables used to manage colors and fonts consistently across the site.
Challenges
Responsive layout was the main challenge, as the arrangement of elements differs between tablet and mobile screen sizes. This was handled using Bootstrap’s Grid system combined with media queries. The <picture> tag with its srcset and sizes attributes was also unfamiliar at first, and required some time to implement correctly for serving different image sizes across screen sizes.
What I Learned
Gained a better understanding of Bootstrap Grid’s responsive logic, learned to use CSS variables to manage a design system consistently, and became comfortable using the <picture> tag correctly for responsive images.
